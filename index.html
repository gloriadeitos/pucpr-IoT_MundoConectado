💻IoT Project - Monitoramento Inteligente

Sistema de Monitoramento Inteligente para Aquários: Tecnologia a Serviço da Vida Aquática

A manutenção de aquários exige atenção constante à temperatura da água, pois variações bruscas podem comprometer a saúde dos peixes. Pensando nisso, desenvolvi um sistema inteligente de monitoramento utilizando Internet das Coisas que integra sensores, microcontrolador ESP32 e interface remota via Blynk.

Dessa forma, o projeto oferece uma solução prática e acessível para aquaristas, permitindo o acompanhamento em tempo real das condições do aquário através de alertas visuais imediatos. Consequentemente, esta implementação demonstra como a tecnologia pode ser aplicada para resolver problemas reais de forma simples e eficaz.

🔧 Componentes do Sistema:
  • Microcontrolador ESP32
  • Sensor de temperatura DS18B20
  • Plataforma Blynk para monitoramento remoto
  • LEDs indicadores de status
  • Alimentação via USB
